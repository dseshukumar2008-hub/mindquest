MINDQUEST 

Mindquest is a small arcade project where I built a collection of 8 mini games to test different skills like memory,speed,logic and focus.
Type the username and select the avatar that you would like to choose in the login page , 

My project mindquest contains total 8 games which includes:
1. Mindquest - A interactive quiz with multiple categories , in the mind quest game you can choose the category and you can also select difficulty level ,
after selecting the difficulty level ,you will be asked a question based on  the category you choosed  with four options and to make it easy i had implemented pause the timer , 
remove the two options , and have to finish the game within 3 chances, each wrong answer will decrease your chance of attempting the quiz.

2. Visual Trap - A memory based game where user needs to remember the emojis , and should answer the questions based on that emojis . And the questions will be like 
what is the first emoji , which emoji is missing and like that.

3. Pattern breaker- This game is a logic based game , where the user will be given patterns and sequences and the user have to find the next number based on sequence .
This game measures users speed and accuracy. User have to quickly identify the missing value or pattern.

4. Reflex Rush - This game is totally a reaction based game . where user have to be very quick . This game contains three modes classic mode , color mode and confusion mode .
This game is very interesting.

5. Memory matrix- A grid based memory game where patterns will be shown for a short time . user needs to remember and select the pattern . As the level increases, thu numberf of
grid boxes increases and would be very tricky.

6. Tap precision - A focus game where user has to click at the right moment . Timing plays a very important role here
   
7. One wrong tile- spot the odd one out . In this game a grid will be shown with similar tiles . User has to choose the tile which is slightly different.

8. Perefct timing - A timing based game where user has to stop the timer close to the target value . This game also includes different modes like easy , medium, hard and 
Trick mode . Where user has to be quick enough.
